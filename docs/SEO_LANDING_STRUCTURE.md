# Путь №1: SEO-лендинг вне SPA

**Рекомендуемая схема:** Vite + React остаются, но лендинг отдаётся как **чистый HTML**, а SPA живёт отдельно. ИИ и поисковики читают полный контент с главной.

---

## Структура

```
/ (главная)
  → index.html  — статический SEO-лендинг (чистый HTML, без React)
  → ИИ и поисковики видят весь текст, тарифы, CTA

/app/ (приложение)
  → app/index.html  — точка входа SPA (React)
  → Вход, регистрация, кабинет, админка
```

### Файлы в проекте

| Путь | Назначение |
|------|------------|
| `index.html` (корень) | Статический лендинг: hero, блоки, тарифы, футер. Стили через Tailwind CDN. Ссылки «Начать» / «Войти» ведут на `/app/#register` и `/app/#login`. |
| `app/index.html` | Точка входа SPA: `<div id="root">` + скрипт `main.jsx`. Мета только для страницы входа/кабинета. |
| `src/` | React-приложение без изменений; при открытии `/app/#login` или `#register` начальный view берётся из hash. |

### Результат сборки (`npm run build`)

```
dist/
  index.html          ← статический лендинг (копируется как есть)
  app/
    index.html        ← SPA, подключает собранные assets
  assets/
    app-*.js
    app-*.css
    ...
  robots.txt
  sitemap.xml
```

---

## Поведение для пользователя

1. Заход на **https://skypath.fun/** — отдаётся статический лендинг (быстро, без JS для контента).
2. Клик «Начать работу» → **/app/#register** — открывается SPA, сразу показывается форма регистрации.
3. Клик «Войти в кабинет» → **/app/#login** — открывается SPA, сразу форма входа.

Хеш `#login` / `#register` обрабатывается в `useView`: при первой загрузке SPA начальный view берётся из `window.location.hash`.

---

## Деплой и сервер

- **/** — отдавать `dist/index.html`.
- **/app**, **/app/** — отдавать `dist/app/index.html` (fallback для SPA, если позже появятся подпути).
- **/assets/** — статика из `dist/assets/`.
- **/robots.txt**, **/sitemap.xml** — из `dist/` (копируются из `public/`).

Пример для nginx:

```nginx
location / {
  try_files $uri $uri/ /index.html;
}
location /app {
  try_files $uri $uri/ /app/index.html;
}
```

## Итог по рендерингу

| Страница | Рендеринг | Для ИИ и поисковиков |
|----------|-----------|----------------------|
| `/` (лендинг) | Статический HTML | ✅ Весь контент в HTML |
| `/app/` (SPA) | Клиентский React | ⚠️ Зависит от выполнения JS |

Главная теперь отдаётся как **чистый HTML** — без риска пустой страницы для краулеров и ИИ.
